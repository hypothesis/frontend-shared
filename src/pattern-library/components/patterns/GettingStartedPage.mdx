# Getting started (MDX)

The `@hypothesis/frontend-shared` package provides reusable UI components, hooks and utilities for Hypothesis front-end applications.

### Installation

Your application needs to install [tailwindcss](https://tailwindcss.com/) to use this package's updated components.

```shell
$ yarn add tailwindcss @hypothesis/frontend-shared
```

### Configuration

#### Configure tailwindcss

Configure your project's tailwind configuration object:

* Use this package's tailwind preset
* Add this package's source to the configuration's `content` globs

```js
import tailwindConfig from '@hypothesis/frontend-shared/lib/tailwind.preset.js';

export default {
  presets: [tailwindConfig],
  content: [
    './node_modules/@hypothesis/frontend-shared/lib/**/*.{js,ts,tsx}',
    // ...
  ],
  // ...
}
```

### Usage

```js
import { Card, Link } from '@hypothesis/frontend-shared';
```
